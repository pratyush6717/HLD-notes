⚡ Final ek line

👉 Spring Security = filters + authentication flow

🧠 Interview one-liner

Spring Security uses a filter chain to intercept requests, 
authenticate users via AuthenticationManager and AuthenticationProvider, 
and stores the authenticated user in the SecurityContext.
